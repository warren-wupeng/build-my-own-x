# 构建你的第一个操作系统 - 学习指南

## 项目概览

你好！我是Claude，你的操作系统开发导师。基于你的背景（5年+编程经验，主要使用Python，希望在1-2周内看到成果），我为你设计了一个渐进式的学习路径。

**你的学习目标**: 构建一个微型操作系统，理解OS核心概念
**目标架构**: x86-64 (资源丰富，适合学习)
**预期时间线**: 分阶段完成，每阶段都有可运行的成果

## 学习路径规划

### 第一阶段：Hello World内核 (第1-2周)
**目标**: 创建能在QEMU中运行的最简内核
- [x] 环境搭建 (交叉编译器、QEMU、调试工具)
- [ ] 编写引导程序 (Bootloader)
- [ ] 创建基础内核，在屏幕上显示 "Hello, OS World!"
- [ ] 理解内核加载过程

**这个阶段结束时你将拥有**: 一个能启动并显示文字的操作系统！

### 第二阶段：内存和中断 (第3-4周)
**目标**: 实现基本的内存管理和中断处理
- [ ] 设置全局描述符表 (GDT)
- [ ] 配置中断描述符表 (IDT)
- [ ] 实现键盘中断处理
- [ ] 基础内存分配器

### 第三阶段：简单Shell (第5-6周)
**目标**: 创建能接受命令的交互界面
- [ ] 基本终端输入输出
- [ ] 命令解析器
- [ ] 实现几个简单命令 (help, echo, clear)

### 第四阶段：进阶功能 (第7-8周及以后)
- [ ] 简单的进程管理
- [ ] 基础文件系统
- [ ] 定时器和任务调度

## 必备知识补强计划

由于你目前对C语言和汇编只有基础了解，我建议同步进行以下学习：

### C语言强化 (与OS开发并行)
- **重点**: 指针、内存管理、位操作、内联汇编
- **推荐资源**:
  - 《C程序设计语言》(K&R) - 经典必读
  - 《C陷阱与缺陷》- 避免常见错误

### 汇编语言入门
- **重点**: x86-64寄存器、基本指令、调用约定
- **推荐资源**:
  - 《Programming from the Ground Up》- 汇编入门
  - 在线参考: https://www.felixcloutier.com/x86/

### 计算机系统基础
- **推荐书籍**:
  - 《深入理解计算机系统》(CSAPP) - 强烈推荐
  - 《操作系统概念》(恐龙书) - 理论基础

## 开发环境设置

### 必需工具
```bash
# 交叉编译工具链
brew install x86_64-elf-gcc  # macOS

# 虚拟机和调试
brew install qemu
brew install gdb

# 构建工具
brew install make
brew install nasm  # 汇编器
```

### 推荐的项目结构
```
os-project/
├── boot/           # 引导相关代码
├── kernel/         # 内核源码
├── include/        # 头文件
├── lib/           # 库文件
├── tools/         # 构建工具
├── docs/          # 你的学习笔记
└── Makefile       # 构建脚本
```
